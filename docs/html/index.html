Purpose¶

Populse_mia, is an ergonomic image processing environment. In this environment, a process pipeline can be easily built by sequentially linking a succession of atomic computation (a brick).

By default, populse_mia offers in its Pipeline Manager (an interface including all the libraries known by populse_mia), two brick libraries, nipype and mia_processes.

One one hand, nipype, an open-source community initiative developed under the umbrella of NiPy is provided as it is and its documentation will be found directly on the nipype website. On the other hand, the mia_processes library developed by the populse team is documented here. Some mia_processes bricks are based on bricks inherited from nipype, others are completely originals.
