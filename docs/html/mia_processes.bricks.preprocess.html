mia_processes.bricks.preprocess package¶

Each package corresponds to a software encapsulated in mia_processes.

Contains:
Packages:

  • ants: bricks from ants

  • dipy: bricks from dipy

  • freesurfer: bricks from freesurfer

  • fsl: bricks from fsl

  • other: bricks from other sources (usually homemade bricks)

  • spm: bricks from spm
